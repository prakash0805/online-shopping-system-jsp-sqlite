Online Shopping System

Project Overview

It is a web-based e-commerce application developed using JSP, HTML, CSS, JavaScript, and SQLite. The application provides a complete online shopping experience where users can browse products, search for items, manage a shopping cart, maintain a wishlist, apply coupon codes, and place orders.

The main objective of this project is to demonstrate the implementation of a dynamic web application using JSP with SQLite database connectivity while providing a modern and user-friendly shopping experience.

Database Tables

Users Table
Stores user registration and login information.

Products Table
Stores product details including product name, price, image, and description.

Cart Table
Stores products added to the shopping cart by users.

Wishlist Table
Stores products saved by users for future purchase.

Orders Table
Stores order information and customer details.

Coupons Table
Stores available coupon codes and discount values.

Application Flow

1. User Registration
New users create an account using the registration page.

2. User Login
Registered users log in using their credentials.

3. Home Page
Users access the home page after successful login.

4. Product Browsing
Users can view all available products.

5. Product Search
Users can search products using the search feature.

6. Product Details
Users can view detailed product information.

7. Add to Cart
Selected products can be added to the shopping cart.

8. Wishlist
Users can save products to their wishlist.

9. Checkout
Users enter delivery information and select a payment method.

10. Coupon Application
Users can apply valid coupon codes to receive discounts.

11. Order Placement
Users confirm the order and complete the purchase process.

12. Order History
Users can view previously placed orders.
